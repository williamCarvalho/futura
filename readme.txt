== Usage ==
To customize the theme navigate to Appearance > Customize.<br />

The landing page is a page template, it can be set to a fixed page in Settings -> Reading, set Front page displays to a static page.<br />

To change certain content on the landing page use the previously created categories:
- Featured Header
- Featured Main
- Featured Parallex
- Featured Content
